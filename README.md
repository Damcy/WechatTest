WechatTest
==========

微信开发测试工具Mac版，a development tool for wechat in Mac


For User:
一个本地和服务器通用的微信后台Mac版测试工具，设计思路模仿Windows下的WinXinPost
用法与WinXinPost相当，填入目标请求地址和选择需要发送的消息类型然后提交即可。
工具为初次开发，测试没有涉及提交强度和超长时间运行，难免有细小Bug难以察觉，请见谅。


———————————————


For Developer：

开发平台：OS X 10.9
开发工具：Qt5.1.1
